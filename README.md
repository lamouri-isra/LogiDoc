# LogiDoc — HOLCIM

Plateforme enterprise de gestion documentaire import pour HOLCIM.

## Démarrage

```bash
npm install
npm run dev
```

Ouvrir **http://localhost:5173/**

## Authentification

- **Inscription** : créez un compte avec email, mot de passe, nom et rôle
- **Connexion** : utilisez les identifiants que vous avez créés
- **Sessions persistantes** : stockage local (localStorage)
- Aucun compte prédéfini — chaque utilisateur crée le sien

## Rôles disponibles

Responsable / Management · Fournisseur · Approvisionnement · Transit · Trésorerie

## Fonctionnalités

- Logo et branding HOLCIM (bleu marine + vert)
- Tableaux de bord professionnels par rôle
- Pilotage & KPI (Responsable / Management uniquement)
- Notifications avec redirection automatique
- Impression réelle (documents, rapports, KPI, PDF)
- Sidebar collapsible · Tables avancées · Animations
